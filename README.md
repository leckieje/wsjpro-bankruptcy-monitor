# WSJ Pro Bankruptcy Risk Score

A weighted composite scoring tool that ranks companies by relative bankruptcy risk using financial indicators from S&P Global Capital IQ.

## How It Works

Each company is scored on up to five indicators — Z-Score, Quick Ratio, Receivables to Revenue, Debt to EBITDA, and FSS Weekly Change — normalized to a 0-100 scale using percentile-clipped min-max normalization (winsorized at p5/p95). The final score is a weighted sum with adjustable weights that must total 100%.

Key scoring features:
- **Outlier handling**: Percentile clipping prevents extreme values from collapsing the scale
- **Missing data**: Weights are redistributed among available indicators per company
- **Debt to EBITDA nuance**: Negative EBITDA = max risk; net cash position = min risk

## Tech Stack

- **Frontend**: React + Vite
- **Data parsing**: PapaParse
- **Data pipeline**: Python (pandas, numpy)
- **Deployment**: GitHub Pages (auto-deploys on push to main)

## Local Development

```bash
npm install
npm run dev
```

## Rebuilding Data

The frontend reads a static CSV (`src/data/sheet.csv`) generated by the Python pipeline:

```bash
cd get-data
pip install -r requirements.txt
python3 build_test_sheet.py --no-fss
```

This merges S&P Global exports with the FSS master list, computes Z-Scores and financial ratios, and outputs the final CSV. Use `--no-fss` to skip the Factiva Sentiment API call.

## Data Sources

- **S&P Global Capital IQ**: Financial fundamentals, credit ratings, market data
- **FSS (Factiva Sentiment Scores)**: Sentiment-based risk signals (optional, requires API key)
